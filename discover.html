Question 1: What are the different ways to select an element in the DOM?

Answer 1: The ways to select DOM elements are by ID, class, tag name, or CSS-style query selectors..

Question 2: What is the difference between innerHTML, innerText, and textContent ?

Answer 2: The difference between innerHTML, innerText, and textContent are given below : innerHTML: Gets or sets the HTML content within an element, including tags.
innerText: Gets or sets the visible text content of an elements.
textContent: Gets or sets all text content of an element including hidden text. .

Question 3:What is event delegation in the DOM?

Answer 3: Event delegation is a technique where a single event listener is attached to a parent element to handle events for all of its descendant elements.

Question 4:What is event bubbling in the DOM?

Answer 4: Event bubbling is when an event on an element "bubbles up" to its parent elements.

Question 5: How do you create, add, and remove elements using JavaScript?

Answer 5: Create: document.createElement('tag')
Add: parent.appendChild(element)
Remove: element.remove()
